# Kharvannan - Professional Portfolio

This repository contains a high-impact, Neo-Brutalist professional portfolio designed to showcase expertise in product growth, revenue engineering, and operations. The website utilizes the "Dewy Morning" color palette, offering a fresh, vibrant, and airy aesthetic with crisp minty off-whites, pure whites, and muted Caribbean green accents.

## 🚀 Website Overview by Section

The portfolio is structured as a continuous, persuasive narrative, guiding the visitor through the professional journey, impact, and operational philosophy of the candidate.

*   **Hero Section:** A bold introduction establishing the core identity and value proposition immediately.
*   **Impact Metrics:** A quick-glance dashboard of key performance indicators (KPIs) demonstrating tangible business results (e.g., revenue supported, churn reduction).
*   **The Approach:** Outlines the core philosophy and pillars of work, emphasizing the bridge between product, marketing, and data.
*   **Operational Framework:** A step-by-step breakdown of the execution strategy, showing exactly *how* results are achieved.
*   **Technical Stack:** A categorized list of tools and technologies mastered, including Analytics, Marketing Automation, and "Vibe Coding" for rapid product prototyping.
*   **Case Studies:** Deep dives into specific business problems ("Fixing the Leaky Bucket" and "Building the Revenue Engine"), detailing the problem, diagnosis, action, and result with visual aids.
*   **Projects & Insights:** A curated archive of thought leadership, guides, and secondary experimental projects.
*   **Academic Foundation & Certifications:** A summary of formal education and professional certifications (e.g., Strategy Consulting, Finance).
*   **Professional Trajectory:** A timeline of career progression and key roles held.
*   **The Final Pitch:** An interactive, accordion-style closing argument addressing "Why me?" and summarizing core strengths.
*   **Footer:** A strong Call-To-Action ("READY TO BUILD & SCALE") with direct contact links (Email, Phone, LinkedIn).

## 🛠️ Built With

*   **React 19** - UI Library
*   **Tailwind CSS** - Utility-first styling (Neo-Brutalist theme)
*   **Framer Motion** - Smooth scroll animations and interactions
*   **Vite** - Lightning-fast build tool
*   **Lucide React** - Crisp, consistent iconography

## 🏃‍♂️ Running Locally

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Build for production: `npm run build`
